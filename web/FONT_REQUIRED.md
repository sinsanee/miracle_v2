# Important: Font File Required

The card generation system requires the Goldman-Bold.ttf font file to render text on cards.

## Setup:
1. Place your `Goldman-Bold.ttf` file in the `web/fonts/` directory
2. If you don't have this font, you can download it from Google Fonts or use an alternative bold font

The font file should be at:
```
web/fonts/Goldman-Bold.ttf
```

Without this font file, card text rendering will fail. Make sure this file is in place before creating cards.
